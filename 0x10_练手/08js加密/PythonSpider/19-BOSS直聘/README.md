# 使用说明

### 注意：我这里只补了两套代码的环境，BOSS实际有几套不知道，补不动了，一天一变...

1.npm需安装express和body-parser，具体自行了解

2.将3个文件放于同一文件夹，cd到该目录，命令行运行
```javascript
node index.js
```

3.运行boss.py开始食用
