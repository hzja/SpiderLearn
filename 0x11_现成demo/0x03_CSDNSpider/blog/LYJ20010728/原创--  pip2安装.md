# pip2安装

#### pip2安装

> 
- 执行下载命令：`curl https://bootstrap.pypa.io/pip/2.7/get-pip.py -o get-pip.py`


> 
- 执行安装命令：`python2 get-pip.py`<br/> 此时在kali上安装的话可能会受到其它python2的pip的影响，直接`python -m pip uninstall pip==报错的版本`即可
