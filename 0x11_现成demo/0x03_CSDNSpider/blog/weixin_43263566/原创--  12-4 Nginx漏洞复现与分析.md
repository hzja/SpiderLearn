# 12-4 Nginx漏洞复现与分析

#### Nginx如何处理PHP请求

Nginx本身不支持直接解析和执行PHP代码，但可以通过与PHP解释器的集成来处理PHP请求。一种常见的方法是使用PHP-FPM（FastCGI Process Manager）作为PHP解释器。

**原理图：**

```
  Step 1                                 Step 2
+---------------------+         +----------------------+
|                     |         |                      |
|     客户端应用程序    |         |   Nginx Web 服务器   |
|                     |         |                      |
+---------------------+         +----------------------+
      |                              |
      |   请求                        |
      +----------------------------→|
      |                              |
  Step 3                                 Step 4
      |                              |
+---------------------+         +----------------------+
|                     |         |                      |
|     PHP-FPM 进程     +-----
```
