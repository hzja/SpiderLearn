# 3-1 Linux网络和系统管理

### 网络管理命令

#### 1、ifconfig 命令

##### 作用

##### 格式

##### 可选项

##### 使用示例

###### 1）显示当前系统上所有活动网络接口的详细信息（包括网卡名字、IP 地址、MAC 地址、状态以及其他统计数据等）

```
ifconfig
```

###### 2）显示指定网卡的信息：

`ifconfig ens33`

###### 3）启动指定网卡：

```
ifconfig ens33 up
```
