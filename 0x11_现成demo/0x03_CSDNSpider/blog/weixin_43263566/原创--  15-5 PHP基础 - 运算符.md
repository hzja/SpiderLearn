# 15-5 PHP基础 - 运算符

### **算术运算符**

|运算符|描述|实例
|------
|+|加法|`$x = 2 + 2; echo $x;`
|-|减法|`$x = 5 - 3; echo $x;`
|*|乘法|`$x = 4 * 3; echo $x;`
|/|除法|`$x = 10 / 2; echo $x;`
|%|取余|`$x = 15 % 4; echo $x;`
|++|自增|`$x = 5; $x++; echo $x;`
|--|自减|`$x = 5; $x--; echo $x;`

#### **算术运算符的使用场景：**

##### 1）加法运算符 +：用于将两个变量相加。

```
&lt;?php
$x = 2; $y = 3;
$result = $x + $y;
echo  $result; // 结果为 5
?&gt;
```
